# Cyber Kill Chain

![](./images/cyber-kill-chain-infographic.jpg)

> Cyber-Kill-Chain（网络攻击链）是一个由Lockheed Martin公司提出的框架，用于描述网络攻击的各个阶段。它最初用于军事领域，但现在被广泛应用于网络安全领域，以帮助识别和防御复杂的网络攻击。Cyber-Kill-Chain模型包括以下七个步骤：

- **侦察（Reconnaissance）**: 在这一步，攻击者收集目标的信息，通常是公开可获取的情报。这可能包括扫描网络端口、社会工程攻击、或者在公开信息中搜索漏洞。攻击者的目的是尽可能多地了解目标环境，以便为下一步行动做准备。

- **武器化（Weaponization）**: 攻击者将收集到的信息用于开发恶意软件、利用漏洞，或者准备钓鱼邮件等攻击手段。这一步通常包括创建一个恶意载荷（payload），并将其包装成攻击媒介，例如通过电子邮件附件或URL链接。

- **交付（Delivery）**: 这是攻击者将恶意载荷传送到目标系统的阶段。常见的方法包括钓鱼邮件、恶意链接、恶意广告、USB设备、甚至是直接的网络攻击。

- **利用（Exploitation）**: 一旦恶意载荷到达目标系统，攻击者会利用系统或应用程序中的漏洞来执行恶意代码。这一步通常包括通过利用已知的安全漏洞获得对系统的访问权限。

- **安装（Installation）**: 在获得系统访问权限后，攻击者会安装恶意软件（如后门程序或远程访问工具），以便持续控制目标系统。这一步使攻击者能够维持对系统的访问，即使原始漏洞被修补。

- **命令与控制（Command and Control, C2）**: 安装后的恶意软件会与攻击者的控制服务器建立通信通道。这允许攻击者远程控制受感染的系统，发出指令，收集数据，或进一步扩展攻击。

- **目标达成（Actions on Objectives）**: 在攻击的最后阶段，攻击者执行其最终目标，这可能包括窃取数据、破坏系统、勒索、破坏服务，或其他恶意活动。攻击者可能会在这一阶段收集敏感信息、加密数据以勒索赎金、或者删除重要文件。
